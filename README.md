# Augmented Reality and Possibilities of Headset Microsoft Hololens 2

Project created as part of Bachelor's thesis. It was created in cooperation with FNU & IT4. Deployed to Microsoft Hololens 2 (UWP). 

Technologies used

	- C++
	- DirectX 11
	- OpenXR
	- CppWinRT
